In order to reliably allow cross-browser input of decimal places, we cannot use the numeric keyboards for fields that require a decimal place. Numeric keyboards on iOS and Android do not include a key for the decimal place. This requires use of type="text" and omitting pattern="[0-9]*". Add data-float to tell the numeric keypress filter to allow decimal marks.
